1 Introduction

Micronaut supports GraphQL via the micronaut-graphql module.

Release History

1.1.0

  • Upgrade to GraphQL Java 13.0

1.0.2

  • Fixed content type resolution (#7).

1.0.1

  • Fixed loading GraphiQL configuration (#6).

1.0.0

  • Removed the GraphQLContextBuilder and GraphQLRootBuilder in favour of a more generic GraphQLExecutionInputCustomizer capable of transforming the ExecutionInput before execution.

1.0.0.RC2

  • Upgrade to GraphQL Java 12.0

  • Introduced a GraphQLJsonSerializer interface and default JacksonGraphQLJsonSerializer implementation capable of providing a customized Jackson ObjectMapper, or plugging in a completely different JSON serializer/deserializer library.

1.0.0.RC1

  • Initial release.
